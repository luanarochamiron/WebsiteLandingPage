# Simple Website Landing Page - CodingLab

Uma Landing Page moderna, limpa e responsiva desenvolvida exclusivamente com HTML5 e CSS3. O projeto simula a página inicial de um site institucional ou portfólio de fotografia, apresentando uma seção de destaque (Hero Section) com imagem de fundo escura, tipografia marcante e botões de chamada para ação (CTA).

## Tecnologias Utilizadas

* HTML5: Estruturação semântica da página, incluindo cabeçalho de navegação e seção principal.
* CSS3: Estilização completa, controle de opacidade, posicionamento absoluto e transições.
* Flexbox: Alinhamento dos links de navegação e centralização dos elementos na seção Hero.

## Funcionalidades e Técnicas Aplicadas

* Barra de Navegação Superior: Menu limpo com links para as seções Home, About, Services, Contact e Feedback.
* Seção Hero Impactante: Uso de uma imagem de fundo escurecida para garantir o contraste ideal com os textos em branco.
* Botões de Chamada para Ação: Botões estilizados ("Learn More" e "Subscribe") com preenchimentos distintos e efeitos visuais ao passar o mouse.
* Tipografia Moderna: Escolha de fontes sem serifa com pesos variados para criar uma hierarquia visual clara no título principal.
